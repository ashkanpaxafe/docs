# Lane Risk Assessment Parameters

<Card title="Configurable Risk Parameters" icon="gauge">
  The following parameters define the core risk assessment metrics for lane validation:

  | Parameter | Description | Default Value |
  |-----------|-------------|---------------|
  | Temperature Range | Acceptable temperature range for shipments | -20°C to 25°C |
  | Transit Time | Expected shipment transit time | 48 hours |
  | Supplier SLA Adherence | Minimum adherence percentage | 95% |
  | Risk Thresholds | Risk scoring thresholds | 20 (low), 50 (medium) |

  <Note>
    These parameters can be adjusted based on specific lane requirements and compliance needs.
  </Note>
</Card>
